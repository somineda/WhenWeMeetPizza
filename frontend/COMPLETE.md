# 🎉 Pizza Scheduler 프론트엔드 완성!

## ✅ 구현 완료된 모든 기능

### 1. 홈페이지 (`/`)
- 서비스 소개 및 주요 기능 안내
- 사용 방법 단계별 설명
- 반응형 디자인

### 2. 인증 시스템
**로그인** (`/login`)
- 이메일/비밀번호 로그인
- 폼 검증 (Zod)
- JWT 토큰 자동 관리
- 로그인 후 "내 이벤트"로 리다이렉트

**회원가입** (`/register`)
- 이메일, 닉네임, 비밀번호 입력
- 비밀번호 확인 검증
- 회원가입 후 자동 로그인
- 이벤트 생성 페이지로 리다이렉트

### 3. 이벤트 관리
**이벤트 생성** (`/events/create`)
- 제목, 설명 입력
- 날짜 범위 선택 (Date Picker)
- 시간 범위 선택 (Time Picker)
- 타임존 선택 (10개 주요 타임존)
- 실시간 폼 검증
- 타임슬롯 자동 생성 안내

**내 이벤트 목록** (`/events/my`)
- 내가 만든 이벤트 카드 리스트
- 참가자 수, 날짜/시간 표시
- 최종 확정 상태 뱃지
- 페이지네이션

### 4. 이벤트 참가
**이벤트 상세** (`/e/[slug]`)
- 이벤트 정보 표시 (날짜, 시간, 타임존, 참가자)
- 공유 링크 복사 기능
- QR 코드 생성 및 표시
- 웹 공유 API 지원
- 최종 확정 시간 강조

**참가자 등록**
- 닉네임/이메일 입력
- 회원/익명 구분
- 자동 인증 감지
- 로컬 저장소에 참가 정보 저장

**가능 시간 선택**
- 날짜별 그룹화된 타임슬롯
- 체크박스 방식 시간 선택
- 전체 선택/해제
- 선택 개수 표시
- 제출 기능

### 5. 대시보드 (`/e/[slug]/dashboard`)
**통계 카드**
- 전체 참가자 수
- 제출 완료/대기 참가자
- 제출률
- 가장 인기 있는 시간

**참가자 현황 테이블**
- 제출 상태 (완료/대기)
- 닉네임, 이메일
- 회원/익명 구분
- 제출한 시간대 개수
- 참가 시각

**히트맵 차트**
- 시간대별 가능 인원 시각화
- 5단계 색상 코드 (0-20%, 20-40%, 40-60%, 60-80%, 80-100%)
- 날짜별 그룹화
- 시간 클릭 시 참가자 목록 표시
- 반응형 그리드 (모바일 2열, 태블릿 4열, 데스크톱 6열)

**최종 시간 선택** (생성자 전용)
- 추천 시간대 상위 5개
- 가능 인원 및 참가자 목록 표시
- 최종 확정 기능
- 확정 이메일 발송

### 6. 공통 컴포넌트
- Button (5가지 variant, 3가지 size, 로딩 상태)
- Input (라벨, 에러, 헬퍼 텍스트)
- Textarea (다중 줄 입력)
- Select (드롭다운)
- Card (Header, Body, Footer)
- Header (반응형 네비게이션)

## 🎨 기술 스택

### 프레임워크 & 라이브러리
- **Next.js 14** - React 프레임워크 (App Router)
- **TypeScript** - 타입 안정성
- **Tailwind CSS** - 유틸리티 CSS 프레임워크

### 상태 관리
- **Zustand** - 경량 상태 관리
- **LocalStorage** - 인증 토큰 영속화

### 폼 & 검증
- **React Hook Form** - 폼 관리
- **Zod** - 스키마 검증

### HTTP & API
- **Axios** - HTTP 클라이언트
- **Interceptors** - 자동 토큰 주입

### UI & UX
- **Lucide React** - 아이콘
- **React Hot Toast** - 토스트 알림
- **QRCode.react** - QR 코드 생성
- **date-fns** - 날짜 포맷팅

## 🔑 주요 기능 상세

## 📞 도움말

### Q: npm install이 실패합니다
**A:** Node.js 버전을 확인하세요 (v18 이상 권장)
```bash
node --version
npm cache clean --force
npm install
```

### Q: 백엔드 연결이 안 됩니다
**A:** 백엔드 서버가 실행 중인지 확인
```bash
curl http://127.0.0.1:8000/api/v1/auth/login/
```

### Q: CORS 에러가 발생합니다
**A:** 백엔드 설정 확인
```python
# config/settings/development.py
CORS_ALLOW_ALL_ORIGINS = True
```

### Q: 페이지가 로딩되지 않습니다
**A:** 브라우저 콘솔(F12) 확인
- 네트워크 탭에서 API 호출 확인
- 콘솔에서 JavaScript 에러 확인
